Installation method description:

Method one: Double click the driver ftdi_ft232_drive.exe to install it,then plug in the device, the computer will automatically identify this device.

Method two: Using the installation method named inf file.This method is a replacement when the method one is invalid.
First, plug in the device to enter the wizard of upadating driver software, the computer will pop up a dialog box of hardware driver installation,then click “browse the computer to find driver software”, and select the installation directory which includes the inf driver folder.
It should be installed twice.

安装方法说明：

推荐方法1

1、使用驱动程序ftdi_ft232_drive.exe，双击安装即可，然后插上设备会自动识别出来。

2、使用inf文件安装方式，适用于方法1无效的用户。
    a)插上设备，执行搜索驱动安装向导，选择目录为inf驱动所在文件夹。
    b)连续安装两次
